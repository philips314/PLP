# I.  
Demostrar ¬(P ⇒ (Q ⇒ P)) FORM.
```
          -----FQ ------FP
          Q FORM  P FORM
------FP -------------- F⇒
P FORM   (Q ⇒ P) FORM
---------------------- F⇒
(P ⇒ (Q ⇒ P)) FORM
--------------- F¬
¬(P ⇒ (Q ⇒ P))
```
