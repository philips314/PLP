## I.  
Demostrar ¬(P ⇒ (Q ⇒ P)) FORM.
```
          -----FQ ------FP
          Q FORM  P FORM
------FP -------------- F⇒
P FORM   (Q ⇒ P) FORM
---------------------- F⇒
(P ⇒ (Q ⇒ P)) FORM
--------------- F¬
¬(P ⇒ (Q ⇒ P))
```
## I. Conjuncion.  
Dar derivacion de P ∧ Q ⊢ Q ∧ P.  
```
-------------ax     -------------ax
P ∧ Q ⊢ P ∧ Q       P ∧ Q ⊢ P ∧ Q
--------------∧e2   ----------------∧e2
P ∧ Q ⊢ Q           P ∧ Q ⊢ P
--------------------------------∧i
P ∧ Q ⊢ Q ∧ P
```
## II. Conjuncion.  
Dar una derivación de P ∧ (Q ∧ R) ⊢ (P ∧ Q) ∧ R
```
                              -------------------------ax
                               P ∧ (Q ∧ R) ⊢ P (Q ∧ R)   
-------------------------ax   ----------------------- ∧e2          --------------------------ax
P ∧ (Q ∧ R) ⊢ P ∧ (Q ∧ R)     P ∧ (Q ∧ R) ⊢ (Q ∧ R)                P ∧ (Q ∧ R) ⊢ P ∧ (Q ∧ R)
-------------------------∧e1  ---------------------∧e1             -----------------------∧e2                                   
P ∧ (Q ∧ R) ⊢ P               P ∧ (Q ∧ R) ⊢ Q                      P ∧ (Q ∧ R) ⊢ (Q ∧ R)
---------------------------------------------∧i                    ---------------------∧e2
P ∧ (Q ∧ R) ⊢ (P ∧ Q)                                              P ∧ (Q ∧ R) ⊢ R
------------------------------------------------------------------------------ ∧i
P ∧ (Q ∧ R) ⊢ (P ∧ Q) ∧ R
```
